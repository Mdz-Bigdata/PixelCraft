How to install
==============
Using pypi
----------
1) Install Python 3
2) Install pixelcraft from pypi using this command ::

    pip install pixelcraft



Install from source
-------------------
1) Install git
2) Install Anaconda or Miniconda Python
3) Open terminal
4) Clone repo from here https://github.com/keplerlab/PixelCraft.git
5) Change the directory to the directory where you have cloned your repo ::

    cd path_to_the_folder_repo_cloned

6) Run the setup ::

      python setup.py install
